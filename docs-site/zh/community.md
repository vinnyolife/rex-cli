# 社区分发

## 渠道组合

- X.com（主阵地，短内容分发）
- GitHub Discussions
- Hacker News / Reddit / Indie Hackers（仅合规发布）

## 每周节奏

1. 周一：发布 1 条战术线程 + 1 条技术片段。
2. 周三：发布 1 条带可量化结果的案例。
3. 周五：发布周复盘并附文档与产品链接。

## 内容规则

- 每条内容只保留 1 个清晰 CTA。
- 一条内容只指向一个目标站点（`cli.rexai.top`、`rexai.top`、`tool.rexai.top` 其一）。
- 避免混合 CTA。

## 安全与审批

- 社交账号发布必须人工审批。
- 先入队草稿，审核后再发。

# Programmatic Page Template

Use this template for scalable landing pages targeting specific long-tail intents.

## Frontmatter

```yaml
---
title: "<keyword-focused title>"
description: "<single intent description>"
---
```

## Structure

1. Problem statement (keyword-aligned)
2. Practical method (step-by-step)
3. Proof or example
4. Internal links (2+)
5. Single CTA destination

## SEO Checklist

- Unique title and meta description
- H1 includes primary keyword naturally
- One clear intent per page
- Canonical URL consistent
- No duplicate CTA destinations in same section

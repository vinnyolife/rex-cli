# Post Templates

## Template A: Launch Update (X Thread)

1. Problem: what users struggled with.
2. Change: what was shipped.
3. Evidence: one measurable result.
4. CTA: one link only.

Example CTA:
- "Try it now: https://cli.rexai.top"

## Template B: Case Study

- Context
- Constraints
- Implementation choices
- Outcomes
- CTA to blog deep-dive (`https://rexai.top`)

## Template C: Tooling Workflow

- Before/after workflow
- Time saved or failure reduced
- CTA to ops tool (`https://tool.rexai.top`)

# Posting Policy

## Scope

This policy controls outbound posting for X and major public communities.

## Rules

1. Do not post unverified performance claims.
2. Do not publish private customer data or session logs.
3. Keep one CTA per post.
4. Keep brand voice factual and non-hype.
5. Use human review before posting from authenticated accounts.

## Quality Checklist

- Problem statement is specific.
- Value proposition is explicit.
- CTA is clear and points to one destination.
- UTM parameters are attached if applicable.

## Escalation

- If content may trigger policy risk, pause and request manual decision.

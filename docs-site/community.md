# Community Distribution

## Channel Stack

- X.com (primary short-form distribution)
- GitHub Discussions
- Hacker News / Reddit / Indie Hackers (policy-compliant posting only)

## Weekly Rhythm

1. Monday: publish one tactical thread and one technical snippet.
2. Wednesday: share a case study with measurable outcomes.
3. Friday: publish recap with links to docs and product destinations.

## Content Rule

- Every post must carry one clear CTA.
- One post = one primary destination (`cli.rexai.top`, `rexai.top`, or `tool.rexai.top`).
- Avoid mixed CTAs in the same post.

## Safety and Approval

- Social account operations remain human-approved.
- Queue drafts first, publish after manual review.

# 프로그래매틱 페이지 템플릿

롱테일 의도를 타겟팅한 랜딩 페이지를 확장 생성할 때 사용하는 템플릿입니다.

## Frontmatter

```yaml
---
title: "<키워드 중심 제목>"
description: "<단일 의도 설명>"
---
```

## 구조

1. 문제 정의 (키워드 정렬)
2. 실전 방법 (단계별)
3. 증거 또는 예시
4. 내부 링크 (2개 이상)
5. 단일 CTA 목적지

## SEO 체크리스트

- 고유한 제목/설명
- H1에 핵심 키워드를 자연스럽게 포함
- 페이지당 단일 의도 유지
- Canonical URL 일관성 유지
- 동일 섹션에서 CTA 목적지 중복/혼합 금지

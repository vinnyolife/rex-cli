# コミュニティ配信

## チャネル構成

- X.com（短文配信の主軸）
- GitHub Discussions
- Hacker News / Reddit / Indie Hackers（ポリシー準拠で投稿）

## 週間リズム

1. 月曜: 戦術スレッド 1 本 + 技術スニペット 1 本
2. 水曜: 計測可能な結果付きケーススタディ
3. 金曜: 週次まとめ + ドキュメント/製品リンク

## コンテンツルール

- 1 投稿 1 CTA を徹底
- 1 投稿 1 目的地（`cli.rexai.top` / `rexai.top` / `tool.rexai.top`）
- CTA の混在は避ける

## 安全性と承認

- 投稿は人手承認を必須化
- 下書きキュー -> 手動レビュー後に公開

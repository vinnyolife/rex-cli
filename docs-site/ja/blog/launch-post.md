# ローンチ: マルチ CLI エージェント向け統一コンテキスト層

多くの AI CLI ワークフローはツール切替時に文脈が失われます。今回のローンチでは Codex / Claude / Gemini をまたいで状態を維持する実用的なコンテキスト層を提供します。

## 変更点

- 共有ファイルシステム ContextDB
- チェックポイントと復旧パケット生成
- 透過ラッピングによる低摩擦導入

## なぜ重要か

クライアントを切り替えてもタスク状態の再構築が不要になります。

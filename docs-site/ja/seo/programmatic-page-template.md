# プログラマティックページテンプレート

ロングテール意図を狙うページをスケール生成するためのテンプレートです。

## Frontmatter

```yaml
---
title: "<キーワード重視タイトル>"
description: "<単一意図の説明>"
---
```

## 構成

1. 課題提示（キーワード整合）
2. 実践手順（ステップ形式）
3. 根拠または事例
4. 内部リンク（2 つ以上）
5. 単一 CTA

## SEO チェック

- タイトルと説明はユニーク
- H1 に主要キーワードを自然に含める
- 1 ページ 1 意図
- Canonical URL 一貫性
- 同セクションで CTA 宛先を混在させない

# Launch: A Unified Context Layer for Multi-CLI Agents

Most AI CLI workflows break when users switch tools. This launch introduces a practical context layer that persists state across Codex, Claude, and Gemini.

## What Changed

- Shared filesystem Context DB
- Session checkpointing and recovery packet generation
- Transparent command wrapping for low-friction adoption

## Why It Matters

Teams no longer need to rebuild task state every time they switch client tools.

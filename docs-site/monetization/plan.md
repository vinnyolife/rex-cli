# Monetization Plan

1. Paid onboarding and workflow setup services
2. Subscription features for operations tooling
3. Enterprise reliability consulting for long-running agent systems

- Top of funnel: docs and social discovery
- Mid-funnel: product exploration on target domains
- Bottom funnel: demo request, paid setup, subscription conversion

1. Free docs and open workflow examples
2. Guided setup package
3. Ongoing support or enterprise engagement

# 即梦AI文生图技能设计

## 1. 概述

创建一个自动化技能，通过浏览器操作即梦AI网站（jimeng.jianying.com），实现文生图功能全自动完成。

## 2. 功能范围

- **核心功能**：文生图（输入中文提示词生成图片）
- **输出**：自动下载保存图片到本地

## 3. 触发方式

- 触发关键词：`即梦生成图片`、`AI生成图片`、`用即梦做图`
- 平台：jimeng.jianying.com

## 4. 自动化流程

```
用户输入提示词
    ↓
打开即梦AI网站 (jimeng.jianying.com/ai-tool/image/generate)
    ↓
检测登录状态 → 未登录则提示扫码
    ↓
输入提示词到文本框
    ↓
选择图片风格（写实/卡通/漫画/插画/唯美）
    ↓
选择图片比例（3:4 竖版 / 1:1 方形）
    ↓
点击生成按钮
    ↓
等待图片生成（最多60秒轮询）
    ↓
下载生成的图片
    ↓
保存到 aios/images/jimeng-{timestamp}.png
    ↓
返回成功消息和图片路径
```

## 5. 技术实现

### 5.1 浏览器操作
- 使用 puppeteer-stealth MCP
- 反检测：随机延迟（3-8秒）、人类行为模拟
- 页面快照获取 DOM 结构

### 5.2 登录检测
- 检测页面是否有登录弹窗或用户头像
- 未登录时提示用户扫码登录

### 5.3 错误处理
- 生成失败：自动重试1次
- 重试仍失败：返回错误信息，提示用户手动操作

## 6. 文件结构

- 技能文件：`memory/skills/即梦AI生成图片.json`
- 输出图片：`aios/images/jimeng-{timestamp}.png`

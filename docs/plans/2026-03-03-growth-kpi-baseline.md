# Growth KPI Baseline

## Baseline Window

- Start date: launch week (Week 0)
- Measurement cadence: weekly review, monthly trend check

## KPI Set

1. Indexed pages (Google/Bing)
2. Organic impressions and CTR
3. Outbound clicks to:
   - `cli.rexai.top`
   - `rexai.top`
   - `tool.rexai.top`
4. Conversion events (trial, contact, waitlist)

## Targets (First 8 Weeks)

- 20+ indexed pages
- CTR > 2.5% on target pages
- 300+ qualified outbound clicks
- 3%-8% destination conversion rate (channel-dependent)

## Reporting Format

- Weekly: wins, losses, next experiments
- Monthly: keep/kill/scale decisions by channel and cluster

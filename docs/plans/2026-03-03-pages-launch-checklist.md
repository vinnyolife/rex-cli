# Pages Launch Checklist

## Build and Deploy

- [ ] `mkdocs build --strict` passes locally.
- [ ] GitHub Actions `docs-pages` workflow passes on `main`.
- [ ] GitHub Pages serves the latest build.

## SEO Readiness

- [ ] `sitemap.xml` available.
- [ ] `robots.txt` available.
- [ ] Search Console verification completed.
- [ ] Bing Webmaster verification completed.
- [ ] Canonical URLs and metadata reviewed.

## Funnel Readiness

- [ ] Home CTA points to `cli.rexai.top`.
- [ ] Blog CTA points to `rexai.top`.
- [ ] Community CTA points to `tool.rexai.top`.
- [ ] Destination links tracked with consistent UTM strategy.

## Post-Launch

- [ ] Publish launch thread on X.
- [ ] Publish one community post.
- [ ] Record Week 1 KPI baseline.

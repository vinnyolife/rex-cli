# Growth Experiment Backlog

## Prioritization

Use ICE (Impact, Confidence, Ease) scoring from 1-10.

## Experiments

1. Homepage CTA wording A/B test.
2. Comparison article title variants.
3. Internal-link density increase in top pages.
4. CTA position test (above fold vs end of article).
5. X thread format test (narrative vs checklist).
6. Landing page with explicit social proof block.

## Success Criteria

- Improvements in CTR, outbound click-through, or conversion.
- If no signal in two cycles, archive experiment.
